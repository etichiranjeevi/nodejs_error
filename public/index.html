Shopping App

API links (will open JSON):

  • /products (list all)
  • /cart (view cart)
  • /payments (view payments)

Try POSTs with curl:

curl -X POST http://localhost:3000/cart -H "Content-Type: application/json" -d '{"productId":1,"quantity":2}'
curl -X POST http://localhost:3000/payments -H "Content-Type: application/json" -d '{"cartId":1,"amount":2000,"method":"card"}'
